Projects - EECS 281

Pathway Optimization

Implemented Prim’s and Kruskal’s algorithms to identify the shortest pathway between given coordinate points.


Three-Dimensional Puzzle

Applied data structures like deques, stacks, and queues in order to implement a backtracing algorithm to find an endpoint given an arbitrary starting point in three-dimensional puzzles (sizes ranging from 4 tile spaces to 7 million). Created and modified custom data structures to allow for multiple modes of both input and output (coordinate list and map).


Priority Queues & Pairing Heaps

This project was meant to simulate a video game in which the player (the user) is supposed to choose the highest-priority enemy to defeat. This was done with the use of priority queues and custom comparators. Additionally, the player could specify output modes that would detail previous player actions and enemy statistics (median lifespan, longest-living enemies, etc.) by implementing the aforementioned data structures in other ways.


Logfile Management

Utilized hash tables to create a tool for storing, sorting, and generating log file reports. This project emphasized algorithmic runtime efficiency as well as taking advantage of the C++ STL in order to create the simplest, quickest, and most elegant solution.
